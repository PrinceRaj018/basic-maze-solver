# 🧩 Basic Maze Solver

A beginner Python project that demonstrates the core concepts behind maze-solving using recursion.

## 🚀 Features

* Maze representation using a 2D list
* Wall detection
* Path detection
* Goal detection
* Recursive movement
* Boundary checking

## 🧠 Concepts Used

* Functions
* Recursion
* Conditions
* Lists
* 2D Arrays
* Problem Solving

## 📚 How It Works

The maze is represented using a 2D list.

* `0` represents a path
* `1` represents a wall

The program:

1. Checks whether the current position is a wall or a path.
2. Uses recursion to move through the maze.
3. Stops when the goal position is reached.
4. Uses boundary checking to prevent invalid movement.

## Example Maze

0 0 0

0 1 0

0 0 0

Goal Position:

(2, 0)
